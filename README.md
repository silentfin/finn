# finn
a static site generator in python

---

##  Getting Started
1. Clone the repository:
    ```bash
    git clone https://github.com/silentfin/finn.git
    cd finn
    ```

2. Make script executable:  
    ```bash
    chmod +x main.sh
    ```

3. Run the script for dev:
    ```bash
    ./main.sh
    ```

4. Visit this url (change the port if you used something else): http://0.0.0.0:8888/
